## 📂 Projects

### Week 1 — Onboarding & Setup
> **Folder:** `week1-onboarding/`

Introduction to the internship environment, tools, and expectations. Includes a personal profile file and a reflection on the onboarding process.

### Week 2 — Student Performance Analysis
> **Folder:** `week2-student-performance/`

Exploratory Data Analysis (EDA) on a student performance dataset. Covers data cleaning, statistical summaries, visualisation of score distributions, and identification of key factors affecting student outcomes.

### Week 3 — E-Commerce Sales Dashboard
> **Folder:** `week3-ecommerce-dashboard/`

Interactive Streamlit dashboard analysing e-commerce transaction data. Features revenue trends, top products, customer segmentation, and actionable business insights.

### Week 4 — Customer Churn Prediction
> **Folder:** `week4-customer-churn/`

End-to-end machine learning pipeline for predicting customer churn. Includes data preprocessing, feature engineering, model comparison (Logistic Regression, Random Forest, XGBoost), and evaluation using accuracy, precision, recall, and ROC-AUC.
